    - [x] マイナスの場合エラー

      -[x] バージョンオブジェクト生成時に patch に -1 を与えると例外が発生する
    - [x] 文字列の場合エラー

      -[x] バージョンオブジェクト生成時に miner に '1' を与えると例外が発生する
      -[x] バージョンオブジェクト生成時に patch に '1' を与えると例外が発生する

      -[x] バージョンオブジェクト生成時に miner に nil を与えると例外が発生する
